# Personal Portfolio Website

A full-stack personal portfolio website developed to showcase projects, skills, resume, and contact information with frontend, backend, and database integration.

---

## Features

* Responsive modern UI
* Home section
* About section
* Skills section
* Projects section
* Contact form
* Resume download option
* Dynamic project management
* REST API integration
* Database connectivity
* Full-stack deployment

---

## Installation

### Clone Repository

```bash
git clone <repository-link>
```

### Frontend Setup

```bash
cd client
npm install
npm start
```

### Backend Setup

```bash
cd server
npm install
node server.js
```

---

## Required Packages

```bash
npm install express mongoose cors dotenv axios
npm install nodemon --save-dev
```

---

## Database Setup

Connect MongoDB using:

```javascript
mongoose.connect('YOUR_MONGODB_URL')
```

## API Endpoints

| Method | Endpoint          | Description        |
| ------ | ----------------- | ------------------ |
| GET    | /api/projects     | Fetch all projects |
| POST   | /api/projects     | Add new project    |
| DELETE | /api/projects/:id | Delete project     |
